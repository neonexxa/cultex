# Basic Webrtc stream and realtime chat 

App title : Tadhacks'17 KL - CULTEX (Cultural Exchange)
Version: 1.0

## Requirement:
- Socket.io (for chat)
- Vidyo.io (for WebRTC video stream)

## Dependencies and packages: 
- Nodejs
- Expressjs
- jQuery
- Browser compatibility WebRTC
- Material Design Bootstrap
